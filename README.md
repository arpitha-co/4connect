🤖 AI-Driven Autonomous Robotic Manipulation System

An end-to-end intelligent automation pipeline that combines AI-powered computer vision, autonomous decision-making, and real-time robotic control using a collaborative robotic arm from Niryo (Ned / Ned2).

📌 Project Overview

In this project, I developed an AI-based computer vision pipeline capable of:

Detecting and classifying objects
Automatically extracting relevant Regions of Interest (ROI)
Making autonomous decisions based on perception
Controlling a robotic arm in real time for precise interaction

The system analyzes the scene, processes visual data, makes independent decisions, and executes robotic manipulation — forming a seamless perception-to-action loop.

🧠 System Architecture

Perception → ROI Extraction → Classification → Decision Logic → Robot Control → Physical Interaction

This project integrates:

🔹 AI-powered object detection & classification
🔹 Automatic ROI extraction and preprocessing
🔹 Autonomous strategic decision-making
🔹 Real-time robot control
🔹 Full end-to-end pipeline from perception to manipulation

🎯 ROI Extraction & Vision Pipeline

The perception module:

Detects objects in real time
Automatically extracts relevant Regions of Interest (ROI)
Applies preprocessing for robust classification
Filters noise and irrelevant background
Passes structured data to the decision engine
This ensures efficient downstream processing and improved decision accuracy.

🦾 Robotic Manipulation (Niryo Ned / Ned2)

The AI module is integrated in real time with the robotic arm:

Scene understanding
Autonomous action selection
Precise motion control
Object interaction and manipulation
Continuous feedback loop

The robot executes decisions generated by the AI system without human intervention.
